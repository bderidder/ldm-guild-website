Recently added events

Events that have been added in the last 7 days and haven't passed yet. On selection of an event, for which the user hasn't signed up yet, the user navigates to a sign up page: Sign Up. Guests can only see the events, they cannot sign up.

My sign ups

Guests cannot see this section at all. If the user is not a guest, a list of sign ups is shown. On selection of a sign up, the user navigates to a page with more detail: View Sign Up.
