Events - La Danse Macabre

Overview of events

  • Guest. A guest can see this page but the actions 'Sign Up' and 'Cancel sign up' are not available.
  • Member. Any member can see this page and perform any actions. The action 'Edit event' or 'Delete event' is only available if the visitor is the organiser of that event.
